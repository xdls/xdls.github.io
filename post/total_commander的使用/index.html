Total Commander 简称TC，是一款应用于 Windows 平台的文件管理器 ，它包含两个并排的窗口，这种设计可以让用户方便地对不同位置的“文件或文件夹”进行操作，例如复制、移动、删除、比较等，相对 Windows 资源管理器而言多出一些功能1。

它拥有文件快速预览、快速搜索、多标签、文件比较、批量重命名、FTP 客户端等诸多实用的功能，并可通过大量的插件进行个性化配置。

需要相应功能的时候，再去寻找插件和下载安装。

安装插件

选项-插件-配置，添加插件

快速搜索插件

QuickSearch eXtended - Total Commander (ghisler.ch)

可以直接使用拼音首字母进行搜索

直接双击压缩包进行安装

按C-s进行搜索

更改设置，直接用字母就可以搜索

配置-选项-快速搜索-字母（打开快速搜索框）

查看器插件

SynPlus 2.7.3 - Total Commander (totalcmd.net)

用来查看JS文件，有代码高亮高亮

HTMLView 1.1 - Total Commander (totalcmd.net)

用来查看文件夹中的内容

sLister 1.1.2 - Total Commander (totalcmd.net)

查看pdf文件

TC插件：用uLister快速查看docx/office2007/2010文件 – 善用佳软 (xbeta.info)

查看多种文件。

WLX Markdown Viewer 2.3 - Total Commander (totalcmd.net)

查看markdown文件

JSON Viewer 1.3.1 - Total Commander (totalcmd.net)

查看json文件
